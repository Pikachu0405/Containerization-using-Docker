# Containerization-using-Docker

1. Create a customised Docker image using Dockerfile

2. Perform data persistence in Docker using volumes and copy command

3. Create a Docker image tar file and extract back the image from the tar file 



**Project Implementation and Takeaways --**

1. Creation of a customised Docker image for a python application using Dockerfile

2. Performing data persistence in Docker using Docker volumes

3. Performing data copy from a Docker container to the Docker host.

4. Creation of an image tar file from the Docker image and also retrieve back image from the tar file

5. Creation of an image tar file directly from Docker container and also retrieve back image from the tar file
